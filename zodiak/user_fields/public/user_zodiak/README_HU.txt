-=LEÍRÁS=-

Figyelem a program a PHP-Fussion 9.xx.xx verzióval kompatibilis!!!!
alacsonyabb verziónál hibák lesznek!!

A Felhasználó profiljába be lehet állítani a felhasználó csillagjegyét, melyet kötelezõvé lehet tenni.


-=TELEPÍTÉS=-



1. Töltsd fel a files mappa tartalmát a szerverre.

2. A böngészõdben nyisd meg a weboldaladat, lépj be fõadminisztrátorként.

3. Nyisd meg az Adminisztrációs oldalt, majd a Felhasználók kezelése lapfülnél válaszd a Profilmezõket,
aktiváld a Csillagjegy profilt, aktiválásnál kiválaszthatod hogy kötelezõ legyen e a kitöltése avagy sem.
